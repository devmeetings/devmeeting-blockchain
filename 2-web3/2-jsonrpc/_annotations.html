Czym jest JSON-RPC

JSON-RPC to prosty bez-stanowy i niezależny od transpotu protokół wywoływania zdalnych metod (Remote Procedure).

Przykład żądania i odpowiedzi:

{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "subtract",
  "params": [{"minuend": 42, "subtrahend": 23}, 5]
}
  
{
  "jsonrpc": "2.0",
  "result": 19,
  "id": 3
}
